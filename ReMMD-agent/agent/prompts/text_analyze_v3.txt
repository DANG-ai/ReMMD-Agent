LABEL DEFINITIONS
================================================================================

T1 Fabrication — text contains content with NO real-world referent (invented facts, quotes, people, specific numbers, attributions that the evidence never corroborates anywhere).
  T1 YES when ANY of these:
    • A SPECIFIC number / dollar amount / percentage / casualty figure that the evidence does not contain at all (e.g. "$10B contract", "3小时小订18000+", "deployed 200,000 troops"); evidence has no such number even approximately.
    • A SPECIFIC quote attributed to a real person where the evidence has nothing matching that quote.
    • A SPECIFIC sub-event ("an internal email said X", "the official report stated Y", "drone footage showed Z") that the evidence does not corroborate.
    • An invented person, organisation, agreement, project name, or contract name.
    • A high-precision figure that suggests fact-grounding (decimal points, exact totals) when evidence has only general or no figures.
    • Post claims "officially confirmed / 经官方证实 / 现场大屏 / 内部邮件" sourcing for facts that the evidence shows came from unverified social media.
  T1 NO when:
    • Post reports general events that the evidence broadly confirms, even with adjectives.
    • Post quotes a person and the evidence ALSO mentions a similar quote (paraphrase counts).
    • Numerical figures match evidence within reasonable rounding (e.g. post "10%" vs. evidence "8.5%" — that is T2, NOT T1).

T2 Distortion — text has a REAL factual basis but is MATERIALLY OVERSTATED / dramatised / mis-attributed / over-generalised / has its conclusion-direction shifted.
  T2 YES when you can name a SPECIFIC sub-claim X where:
    Evidence says X happened with magnitude / scope / consequence A;
    Post says X happened with magnitude / scope / consequence B; AND
    B is materially STRONGER / more dramatic / more conclusive than A.
  Concrete T2 patterns:
    • Adjective inflation: "surge / crisis / panic / decisive victory / on the brink / catastrophic" when evidence says "small uptick / loss / concern / stopgap / measured / routine".
    • Magnitude inflation: "tens of thousands died" when evidence says "thousands" or "unspecified"; "$10B contract" when evidence has "billions".
    • Conclusion extrapolation: data shows "core PCE remains above 3%" but post writes "通胀压力再度升温 / 市场更趋谨慎 / 美联储或将更趋鹰派".
    • Speculation framed as fact: "X will surely cause Y" when evidence only shows "X happened".
    • Source upgrade: "officially confirmed" when evidence says "according to officials familiar with the matter".
    • Drawing a "decisive / clear / complete / unprecedented" interpretation when underlying facts are ambiguous, partial, or routine.
    • Narrative inflation: stitching multiple unrelated real events into a unified "crisis is escalating" / "long war coming" narrative when evidence's tone is measured / event-by-event neutral.
  T2 NO when:
    • Post is plain factual reporting that closely matches evidence's tone AND certainty.
    • Post quotes a person's prediction/opinion as a quote attributed to that person — that is reporting, not distortion.
    • Post uses standard journalistic phrasing ("key", "important", "significant") that does not materially exaggerate.
    • Minor / cosmetic framing differences without a meaningful magnitude/scope/conclusion shift.
    • Past events correctly anchored to their historical date even if evidence is dated later.

T3 Misleading Context — text has REAL content but placed into a WRONG temporal / spatial / event-frame / source / attribution context.
  T3 YES when:
    • Post says "today" / "this week" but evidence places the event in a MATERIALLY different time period.
    • Post claims event in country/region X but evidence locates it in Y.
    • Post attributes a real quote to person A but evidence says person B said it.
    • Post uses real archival imagery (per image descriptions) but presents it as live / recent.
    • Post takes a historical quote/decision and presents it as current.
    • Post's claimed source ("according to AP/Reuters/官方/IDF") differs from what evidence actually shows.
    • IMAGE-FROM-DIFFERENT-EVENT: post text describes event A and uses an image that depicts a real different event B (per the image descriptions you receive).
  T3 NO when:
    • Date discrepancies are minor (1-3 days off) and don't change the meaning.
    • Post simply describes events that evidence also describes in similar terms.
    • Past events correctly anchored to their historical date.
    • Local events reported as local with proper geographic specificity.

C1 Semantic Inconsistency — TEXT and IMAGE disagree on entity / event / attribute (use the image descriptions provided).
  C1 YES when:
    • Post text says "Präsident X signed", image description shows "Präsident Y".
    • Post text says "explosion at factory A", image description shows "smiling family at beach".
    • Post text says "image shows the rocket launch", image description shows "missile booster on the ground".
    • Post text references a number / value that DIFFERS from the same number visible in the image OCR.

Base rates (500 samples; about 80% of posts carry ≥ 1 textual issue):

STEP 0 — Read the POST TEXT and EVIDENCE side by side. Decide an INTUITIVE alignment level:
   • WELL_ALIGNED: post's facts, tone, certainty, attribution, time/place all match evidence; no concrete distortion.
   • PARTIALLY_ALIGNED: post matches evidence on most facts but with some overstatement / context drift / minor mismatch.
   • MISALIGNED: post materially differs from evidence (different facts, much stronger framing, wrong context, etc.).

STEP 1 — Apply the labels:
   • If WELL_ALIGNED ⇒ T1=no, T2=no, T3=no, C1=no — ALL FALSE. The 20% TRUE posts are well-aligned and MUST come back with all NO.
   • If PARTIALLY_ALIGNED ⇒ usually 1-2 of {T2, T3} fire. Adjective inflation / dramatic framing → T2; wrong time-place / image-from-different-event → T3.
   • If MISALIGNED ⇒ usually 2-3 of {T1, T2, T3} fire — name them concretely.

For C1: decide independently by comparing image descriptions to text claims. Whenever the IMAGE'S entities/objects/event-content semantically disagree with the post text's claim, flag C1.

For each YES, you MUST be able to write a SPECIFIC sentence naming what triggers it (citing evidence_id or image filename). Vague feelings are NOT enough — a concrete sentence IS enough.

DECISION FIRMNESS — DO NOT LOOP:

- If evidence partially supports a claim (topic correct but magnitude/scope/conclusion overstated), that is T2, NOT T1.

NEVER:
- NEVER flag T1 / T2 / T3 / C1 for a post where evidence broadly matches and only adjectives differ.
- NEVER label any T-label as YES when you cannot write a CONCRETE one-sentence evidence string.
- NEVER write reasoning inside the JSON fields. The JSON must be pure structured output.
