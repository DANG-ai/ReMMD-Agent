一级标签（Verdict）：5-Point Scale
1. True
定义：
样本中的 核心表达（core claim）成立，且其支撑性细节、限定条件和语境信息均与可验证事实一致；不存在会改变受众整体理解的错误、遗漏或错置。
判定标准：
•	核心 claim 完全成立。 
•	关键细节准确。 
•	不存在实质性的语境错置。 
•	不存在会改变结论方向的遗漏或误导。 
操作性边界：
只要存在任何会改变核心理解的失真，就不应标为 True。
这和 PolitiFact 对 True 的思路一致：陈述准确，且没有重要缺失。

2. Mostly True
定义：
样本中的 核心表达成立，整体结论方向正确，但存在局部性、不关键的事实偏差、细节错误、限定不充分或轻度语境偏差；这些问题不会实质性改变受众对核心 claim 的理解和最终判断。
判定标准：
•	核心 claim 成立。 
•	存在细节级问题，但不改变主要结论。 
•	错误不影响样本的总体真实性方向。 
•	若去除这些瑕疵，样本表达的主要判断仍然成立。 
操作性边界：
Mostly True 的关键不是“有一点错”，而是：
错的部分不改变主结论。
与 Mixture 的边界：
若错误或遗漏已经足以让受众形成显著偏差理解，即使核心 claim 有部分真实基础，也不应再标 Mostly True，而应进入 Mixture。
这也接近 PolitiFact 对 Mostly True 的用法：整体准确，但需要澄清或补充。
________________________________________
3. Mixture
定义：
样本同时包含重要的真实成分与重要的虚假、误导或错置成分；其核心表达不能被简单地判定为总体真实或总体虚假，因为样本的真实性与误导性在整体判断中都具有实质权重。
判定标准：
满足以下任一情形即可：
•	核心 claim 只在部分条件下成立，但被表达为无条件成立； 
•	样本中既有关键真实信息，也有关键虚假信息，二者共同决定整体理解； 
•	样本的主要误导来自断章取义、脱离语境、重新配对、错误背景化； 
•	如果只看“真实成分”会高估其真实性，只看“虚假成分”又会低估其真实性。 
操作性边界：
Mixture 不是“轻微瑕疵”，也不是“基本错误”；
它表示：真假两部分都重要，缺一方都无法准确描述样本。
与 Mostly True 的边界：
如果真实部分占主导，且错误不改变主结论，标 Mostly True；
如果错误已经足以显著改变理解，但又不能说核心上总体为假，标 Mixture。
与 Mostly False 的边界：
如果虚假/误导部分已经主导整体印象，使受众得到的主要结论方向是错的，则应标 Mostly False；
如果真假两部分在整体判断中都不可忽略，标 Mixture。
Snopes 对 Mixture 的定义就是：包含显著的真与假成分，以至于不能公平归入其他标签；而 PolitiFact 的 Half True 也强调“部分准确，但遗漏关键细节或脱离上下文”。你的 Mixture 实际上可以视作这两种思想的结合。

4. Mostly False
定义：
样本中的 核心表达总体上不成立，或样本传达给受众的主要印象/主要结论是错误的；但其中仍包含少量真实元素、真实片段或局部正确细节。
判定标准：
•	核心 claim 总体不成立，或其结论方向错误； 
•	真实元素存在，但不构成主要判断依据； 
•	即使移除其中真实元素，样本的主要误导性仍然存在； 
•	真实成分不足以挽回整体判断。 
操作性边界：
Mostly False 的关键不是“有真成分”，而是：
真成分是次要的，假或误导是主导的。
与 Mixture 的边界：
如果去掉虚假部分后，样本还能保留一个相当重要的真实核心，偏 Mixture；
如果去掉真实部分后，样本的核心误导几乎原封不动地保留，偏 Mostly False。
这与 PolitiFact 对 Mostly False 的思路一致：有一点真实成分，但忽略关键事实后给出了不同甚至相反的印象。
________________________________________
5. False
定义：
样本中的核心表达不成立，且其主要事实内容缺乏真实对应基础；不存在足以维持其核心结论的真实支撑。
换言之，样本所传达的中心命题整体上是捏造的、无事实依据的，或与事实明确相反。
判定标准：
•	核心 claim 不成立； 
•	样本缺乏维持核心 claim 的真实基础； 
•	即使存在零散真实元素，这些元素也不构成对核心表达的实质支持； 
•	样本的中心结论可明确判定为错误。 
操作性边界：
False 对应的是：
核心命题整体失效。
与 Mostly False 的边界：
Mostly False 仍保留某些可识别的真实支撑成分；
False 则表示这些真实成分不足以被视为核心 claim 的组成支撑，或核心命题本身即无真实对应。
