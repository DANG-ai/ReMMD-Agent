# ReMMD-Agent Method Description

本文方法将多模态虚假信息检测建模为一个“原子化分解-证据检索-结构化裁决”的推理流程。给定一条包含文本和若干图像的社交媒体或新闻帖文，系统并不直接要求模型输出真假标签，而是先将帖文拆解为可验证的最小事实单元，再围绕这些事实单元检索外部证据，最后由多模态大模型在原帖、原子事实、检索证据和辅助分析结果的共同约束下完成三层判定。这样的设计将端到端分类问题转化为更接近人工事实核查的过程：先明确“需要核查什么”，再收集“可以支持或反驳这些内容的证据”，最后进行“证据对齐后的综合判断”。

## Atomic Point Parsing

第一阶段是原子点解析。该阶段使用视觉语言模型对输入帖文进行一次结构化的多模态问答，将原始帖文转化为层次化的 atomic points。这里的 atomic point 指的是可以被独立验证的最小事实或语义单元，例如某个实体、时间、地点、数量、归因、图像中可见对象，或文本与图像之间的绑定关系。与直接对整篇帖文进行检索不同，原子点解析显式地把复杂叙事拆成多个短小、具体、可检索的核查目标。

该阶段的输出覆盖四类互补信息。第一类是图像层面的观察，描述每张图像客观呈现的场景、人物、物体、文字、标注、截图样式或其他视觉元素。第二类是跨模态绑定，描述帖文如何使用图像来支撑文本中的某个事件、人物、地点、时间、数字或结论。第三类是句子层面的文本原子点，将每个句子中的复合主张拆解为若干独立事实。第四类是段落或帖文层面的整体主张，捕捉跨句子的总结性叙事、因果框架和核心结论。

除了事实原子点，解析阶段还生成面向检索的查询表达和结构化视觉线索。检索查询通常包含实体、事件、地点、日期、组织名称、数字、引语和可能的别名或转写形式；视觉线索则记录每张图像的场景描述、OCR 文本、可识别实体、覆盖文字、箭头、圆圈、横幅、水印等图像元素。这些视觉线索本身不直接给出失真类型标签，而是作为后续检索和裁决的中间表征。这样可以避免在早期阶段过早分类，同时保留足够细粒度的视觉证据。

从论文方法角度看，atomic point parsing 的核心作用是构建一个 retrieval-oriented representation。它将多模态帖文从非结构化输入转化为一组可索引、可比对、可追踪的事实探针。相比直接使用整篇文本或整组图像进行检索，这种分解能够提高证据召回率，尤其适用于包含多个实体、多个时间地点、多个图像以及复杂图文关系的 misinformation 样本。

## Memory-Augmented Evidence Retrieval

第二阶段基于原子点进行证据检索。系统维护一个样本级 memory bank，其中包含与每个待检测样本相关的异构证据，例如新闻片段、事实简报、事实核查材料、社交媒体材料，以及对参考图像或真实图像内容的文字描述。检索时，每个 atomic point 和检索查询都会被映射到语义向量空间，并在对应样本的 memory bank 中寻找最相关的证据。

这种检索策略具有两个特点。首先，检索是 atomic-point-driven 的：每个最小事实单元都可以独立触发证据查找，因此系统不依赖帖文整体语义与证据整体语义之间的粗粒度匹配。其次，检索是 high-recall oriented 的：系统倾向于保留较宽泛的候选证据，并在最终裁决阶段交由大模型判断哪些证据真正支持、部分支持、反驳或无法验证某个子主张。这样的设计降低了早期检索漏掉关键证据的风险。

对于多模态虚假信息检测，memory bank 中的图像内容描述尤其重要。它为模型提供了一个可与帖文图像进行对照的参考视角：如果帖文图像与证据中的图像描述不一致，系统可以进一步判断是否存在图像来源错误、图像被编辑、图像与文本语义不匹配，或图像被置于错误事件、地点、时间和语境中。

## Auxiliary Textual and Visual Analysis

在最终裁决前，系统还引入了面向失真类型的辅助分析。文本分析模块聚焦于文本事实是否存在编造、夸大、语境误置、归因错误或文本-证据语义不一致；视觉分析模块聚焦于图像是否可能为合成内容、是否存在可见编辑、以及图像与文本在实体、事件、语境和立场上的一致性。这些模块并不直接决定最终输出，而是为最终裁决提供 soft priors。

这种设计体现了 multi-expert prompting 的思想：不同子任务由不同的专门视角进行分析，最终由统一的裁决器进行融合。辅助分析可以提高模型对细粒度失真模式的敏感度，尤其是在视觉编辑、合成图像、图像错配、文本夸张和语境误导等容易被单次整体分类忽略的场景中。同时，最终裁决仍需要重新检查原帖和证据，因此辅助分析结果只是可被采纳或被证据推翻的中间判断。

## Final Judge

最终阶段是 evidence-grounded multimodal judgement。该阶段再次将原始文本、原始图像、原子点表示、检索证据、搜索结果以及辅助文本/视觉分析输入视觉语言模型，并要求模型输出一个结构化的三层判断。第一层是五分类真实性 verdict，覆盖 True、Mostly True、Mixture、Mostly False 和 False。第二层是多标签失真分类，覆盖文本失真、视觉失真和跨模态不一致等类型。第三层是自然语言解释，用于概括帖文为何被判定为对应类别。

最终裁决不是简单投票或规则匹配，而是一个结构化证据融合过程。模型首先需要识别帖文的核心主张，并抽取若干关键子主张；随后逐一检查这些子主张与证据之间的关系，将其标记为 supported、partially supported、contradicted 或 unverified。对于包含图像的样本，模型还需要逐图判断图像是否匹配帖文声称的内容、是否匹配 memory bank 中的图像内容描述、是否存在视觉合成或编辑线索，以及是否出现语义、语境或语用层面的图文不一致。最后，模型基于这些子主张发现和图像发现，选择对应的 L1 verdict、L2 distortion taxonomy 和 L3 rationale。

这一阶段的关键在于将标签预测约束在显式推理结构之内。L1 判定不再是从原帖到标签的黑箱映射，而是建立在子主张证据状态、图像-文本一致性、视觉失真线索和标签定义之间的对齐关系上。L2 标签也不是由单一视觉或文本线索触发，而是由模型在原子点、证据和图像检查的基础上逐类判断。L3 解释则要求以原帖语言给出简洁理由，从而使最终输出保持可读性和可审查性。

## Method-Level Interpretation

整体来看，本方法的核心贡献可以概括为 hierarchical atomic decomposition for multimodal misinformation detection。传统多模态分类方法通常直接将文本和图像输入模型并预测标签，而本方法显式地引入一个中间层：多粒度、可检索、可验证的 atomic representation。该表示同时包含图像观察、文本事实、跨模态绑定和整体叙事，使系统能够围绕具体核查点而不是整篇帖文进行证据收集和推理。

这一中间表示带来了三方面优势。第一，它提高了检索可控性：每个原子事实都能生成明确的证据查询，降低复杂帖文中关键细节被整体语义淹没的风险。第二，它提高了推理透明度：最终判定可以追溯到哪些子主张被支持、哪些被反驳、哪些仍未验证，以及哪些图像线索导致了视觉或跨模态失真标签。第三，它提高了多模态适配性：图像不只是作为分类输入存在，而是被拆解为可核查的视觉事实，并与文本主张建立显式绑定。

因此，ReMMD-Agent 可以被描述为一个 memory-augmented, decomposition-based multimodal reasoning framework。它先通过单轮结构化大模型解析获得原子事实图谱，再通过样本级记忆库检索获得相关证据，最后通过单轮结构化大模型裁决完成 L1/L2/L3 输出。整个流程将大模型的生成能力用于事实分解和证据融合，而将检索模块用于提供可验证的外部依据，从而在多模态 misinformation detection 中兼顾判定性能、可解释性和证据可追踪性。
